# HotelBookingSystem

Sistema de reservas de hotel desarrollado en C#.

## Funcionalidades
- Crear reservas con múltiples habitaciones.
- Asignar tipo de habitación (Standard, Premium, Suite) a toda la reserva.
- Mostrar detalles completos de cada reserva.
- Interfaz de consola limpia y validada.

## Uso
Ejecutar la aplicación y seguir las opciones del menú para crear y listar reservas.
